# Worked examples

End-to-end walkthroughs of the playbook loop, for onboarding teammates.

1. [MCP token bloat](./example-1-mcp-token-bloat.md) — a heavyweight problem with measurement, investigation, fix, and doc.
2. [Skill not triggering](./example-2-skill-not-triggering.md) — a lightweight problem showing the loop still applies to 5-minute fixes.

## The loop (one-line version)

`Problem hit → Issue → Investigate publicly → Fix → Doc PR → Close → Share`

## Rules we follow

- Every problem becomes an issue, even if you think you'll fix it in a minute.
- Evidence lives in issue comments, not Slack.
- Fixes land as docs. No doc, not solved.
- Issues close with a link to the doc.
- When in doubt, write it down.
